# Bot Commands

- **`!commands`**\
  _Alias_: (`!command`, `!cmd`, `!help`)\
  Show link to this page with commands list.

- **`!nickname` <new_nickname>**\
  _Alias_: (`!nick`)\
  _Example:_ `!nickname bananaman`\
  Update user's nickname in Chibi's memory.
